### 5.1 Паспорт № ISCV-2026-0001 — 1.1.1.1 Ensure cramfs kernel module is not available (Automated)

**Тип блока:** несоответствие конфигурации.

| Поле | Значение |
|---|---|
| ID паспорта | ISCV-2026-0001 |
| Название | 1.1.1.1 Ensure cramfs kernel module is not available (Automated) |
| Тип finding | Несоответствие конфигурации |
| Статус | fail |
| Хост | target1 |
| Актив / компонент | не применимо |
| Нормализованный severity | high |
| Категория | configuration |
| Источник | wazuh_sca |
| Дата выявления | данные отсутствуют |
| Rule ID | cis_ubuntu24-04:35500 |

| Поле | Значение |
|---|---|
| Команда проверки | данные отсутствуют |
| Ожидаемый результат | данные отсутствуют |
| Ссылки | данные отсутствуют |

**Описание уязвимости / несоответствия:**  
The cramfs filesystem type is not commonly used and should be disabled.

**Evidence:**  
```text
Command: modprobe -n -v cramfs
Compliance: cis: 1.1.1.1
Rules: c:modprobe -n -v cramfs -> r:^install /bin/false
```

**Возможные последствия:**  
Disabling uncommon filesystems reduces the kernel attack surface.

**Возможные меры по устранению уязвимости:**  
Edit or create a file in `/etc/modprobe.d/` ending in `.conf` and add `install cramfs /bin/false`.

| Дата выявления | данные отсутствуют |

| Поле | Значение |
|---|---|
| Команда проверки | данные отсутствуют |
| Ожидаемый результат | данные отсутствуют |
| Ссылки | данные отсутствуют |

**Описание уязвимости / несоответствия:**  
The SSH daemon should not permit X11 forwarding.

**Evidence:**  
```text
Command: sshd -T
Compliance: cis: 5.1.20
Rules: c:sshd -T -> r:^x11forwarding no$
```

**Возможные последствия:**  
X11 forwarding expands the SSH session attack surface.

**Возможные меры по устранению уязвимости:**  
Set `X11Forwarding no` in sshd configuration and restart SSH.
